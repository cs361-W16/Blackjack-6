Rules of Black Jack

  1. After all the players have dealt their hand, the dealer will show all the cards in their hand.
  2. The dealer then adds up their cards according to the rules. For Aces, the dealer must score them as 11, unless doing so causes them to bust. (lose the hand)
  3. If the dealer's hand is worth less then 17, the dealer will hit
  4. If the dealer's hand is worth 17 or more, the dealer will hold
  5. If at any point the dealer's hand is worth more then 21, then the dealer immediately looses and the player wins.
  6. At the end of the round, any player with more points then the dealer's hand (but both are less then 21) wins.
  7. If they have the same amount of points, then the money stays where it is, and a new game is started.
